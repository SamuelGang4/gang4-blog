Tu as lu le guide stratégique. Tu as convaincu ton responsable. Le creator est partant. Il reste une question : comment tu fais concrètement, dans Ads Manager, pour que cette pub parte depuis son compte à lui ?

C'est l'étape que personne ne détaille vraiment. Soit parce qu'ils supposent que tu sais, soit parce qu'ils confondent encore l'ancien système Branded Content Tool avec le système de codes actuel.

Ce guide couvre uniquement le setup technique. Pas la stratégie, pas le ROI. Juste les étapes, dans l'ordre.

## Comprendre la mécanique avant de cliquer

Avant d'ouvrir Ads Manager, une minute de clarté sur ce qui se passe sous le capot.

Une Partnership Ad, c'est une pub que **tu paies et tu pilotes**, mais qui **s'affiche depuis le compte du créateur**. Pour que Meta accepte ça, il faut une autorisation explicite du créateur. C'est le système de codes.

Il existe deux types de codes :
- **Code LBC** (Linked Business Content). Pour les Partnership Ads sur **Instagram**
- **Code CAA** (Creator Ad Authorization). Pour les Partnership Ads sur **Facebook**

Si tu veux diffuser sur les deux plateformes, le créateur génère deux codes. Ils ne sont pas interchangeables.

Ces codes sont à usage unique et valables 30 jours. Une fois entré dans Ads Manager, le lien est établi. Et aucun échange de mot de passe n'a eu lieu. Le créateur garde le contrôle total de son compte.

<Callout variant="insight" title="Nouveau terme, ancienne logique">
Si tu as déjà travaillé avec les Branded Content Ads (avant le rebranding Meta de 2023), la mécanique est identique. Le nom a changé, l'interface Ads Manager a été simplifiée, et la compatibilité avec les campagnes Advantage+ a été ajoutée. Les codes LBC et CAA existaient déjà sous d'autres noms.
</Callout>

## Ce qu'il faut vérifier avant de commencer

Un setup Partnership Ads rate presque toujours à cause d'un prérequis manqué. Pas à cause d'une erreur dans Ads Manager.

**Côté créateur (à vérifier avant de demander le code) :**
- Compte Instagram en mode **Professionnel** ou **Créateur** (pas Particulier)
- Page Facebook associée si tu veux des PA Facebook
- La fonctionnalité "Partenariats de marque" activée dans les paramètres du compte (elle est disponible sur les comptes professionnels sans seuil de followers minimum, contrairement à une idée reçue)

**Côté marque :**
- Un **Business Manager actif** avec accès aux actifs de l'entreprise
- Un **Pixel Meta** opérationnel (pour la mesure des conversions)
- Le créatif prêt à uploader au format Partnership Ads (vertical 9:16 recommandé, 4:5 accepté)

**Côté contrat :**
- Le [contrat creator](/blog/contrat-creator-partnership-ads-meta-template) signé avant de générer les codes. Pas après. Une fois la pub en ligne, négocier les clauses devient beaucoup plus difficile.

## Étape 1. Le créateur génère son code de permission

C'est l'étape que **tu ne peux pas faire à sa place**. C'est lui qui génère, c'est lui qui te communique le code.

**Sur Instagram (code LBC) :**

1. Le créateur ouvre l'app Instagram
2. Va dans **Paramètres** (icône hamburger en haut à droite > Paramètres et activité)
3. Sélectionne **Créateur** (ou "Compte professionnel" selon la version)
4. Tape sur **Partenariats de marque**
5. Sélectionne **Contrôles publicitaires**
6. Appuie sur **Générer un code**

Le code s'affiche à l'écran. Il est alphanumérique, environ 10 à 15 caractères. Le créateur te l'envoie (SMS, email, DM. Peu importe, c'est juste un code texte).

**Sur Facebook (code CAA) :**

1. Le créateur ouvre Facebook
2. Va sur sa **Page** (pas son profil perso)
3. Accède à **Paramètres de la Page**
4. Sélectionne **Partenariats de marque**
5. Tape sur **Contrôles publicitaires**
6. Génère le code CAA

<Callout variant="tip" title="LBC ≠ CAA : ne pas mélanger">
Le code LBC Instagram ne fonctionne pas sur Facebook, et inversement. C'est l'erreur numéro 1 des équipes qui configurent leurs premières PA. Si tu veux des placements partout sur Meta, demande les deux codes au même moment. Ça évite un deuxième aller-retour avec le créateur.
</Callout>

## Étape 2. Tu entres le code dans Ads Manager

Maintenant tu prends la main. Le créatif est prêt, le code est en ta possession, le contrat est signé.

1. Ouvre **Meta Ads Manager**
2. Crée une nouvelle campagne ou ouvre une campagne existante (ASC ou CBO)
3. Au niveau de l'**ensemble de publicités**, vérifie le placement : Meta Advantage+ ou placement manuel selon ta structure
4. Au niveau de la **publicité** (le créatif), cherche la section **Identité**
5. Dans "Identité", tu verras une option **"Utiliser une identité de partenariat"** ou **"Partnership Ad"** selon la version d'Ads Manager
6. Entre le code LBC (pour Instagram) ou CAA (pour Facebook) dans le champ prévu
7. Meta vérifie le code en temps réel. Si valide, le compte du créateur apparaît comme identité de diffusion

<Callout variant="tip" title="Interface Ads Manager : variable selon les comptes">
Si tu ne trouves pas l'option "Partnership Ad" à l'endroit indiqué, cherche dans la section "Identité" ou "Contenu de partenariat". Meta teste régulièrement des layouts différents selon les comptes et les régions.
</Callout>

## Étape 3. Tu configures le créatif et tu lances

Une fois le code validé, tu travailles dans Ads Manager comme pour n'importe quelle pub.

**Le créatif :**
- Upload la vidéo ou l'image fournie par le créateur
- Ajoute le texte, la headline, la description et le CTA (c'est la marque qui écrit le copy. Pas le créateur)
- Le nom affiché dans le fil d'actualité sera celui du créateur, mais la mention "Partenariat rémunéré avec [Marque]" s'ajoute automatiquement. Conformément à la Loi Billon

**L'injection dans la campagne :**

La recommandation 2026 est claire : **injecte les PA dans ta campagne ASC broad existante**, pas dans une campagne isolée.

Pourquoi ? [Andromeda](/blog/andromeda-meta-algorithme-delivery) a besoin de voir tes Partnership Ads en compétition directe avec tes autres créatifs pour optimiser la distribution du budget. Une campagne PA isolée prive l'algorithme de données comparatives, allonge la phase de learning, et fragmente inutilement ton budget.

L'exception : si tu veux mesurer l'incrémentalité pure des PA. Dans ce cas, une campagne test dédiée sur 7 à 14 jours, à budget constant, te donne une mesure propre. Mais c'est un test. Pas une structure permanente.

**Le lancement :**
- Meta soumet la pub à une revue de modération standard : 24 à 48 heures ouvrées
- Pendant ce temps, **le créateur reçoit une notification** lui demandant de valider le partenariat dans son interface
- La pub ne se diffuse pas tant que cette validation n'est pas faite

C'est l'étape que les marques oublient le plus souvent. Tu peux avoir tout configuré côté Ads Manager. Si le créateur n'a pas cliqué sur "Approuver" dans ses notifications, rien ne part.

## Les 5 erreurs qui bloquent tout

Aucune de ces erreurs n'est grave. Toutes sont évitables.

**Erreur 1 : Demander le code trop tôt**
Le code expire en 30 jours. Si tu le demandes pendant que l'équipe créa finalise le contenu, il peut expirer avant que tu aies fini. Règle simple : demander le code 72 heures maximum avant l'upload prévu.

**Erreur 2 : Utiliser le code LBC sur un placement Facebook**
Codes LBC = Instagram seulement. Codes CAA = Facebook seulement. Toujours vérifier le placement avant d'entrer le code.

**Erreur 3 : Créer une campagne PA isolée**
Le [combo Partnership Ads + Andromeda](/blog/partnership-ads-andromeda-combo-gagnant-2026) ne fonctionne à plein régime que dans une ASC broad avec tous tes créatifs. Campagne isolée = phase de learning allongée + budget fragmenté.

**Erreur 4 : Oublier la validation créateur**
Meta envoie une notification au créateur quand tu soumets la PA. Jusqu'à ce qu'il valide, la pub est en attente. Prévoir dans le process : envoyer un message au créateur le jour du lancement pour lui rappeler de valider.

**Erreur 5 : Lancer sans contrat signé**
Une Partnership Ad sans [contrat créateur](/blog/contrat-creator-partnership-ads-meta-template) bien rédigé, c'est une relation commerciale sans garde-fous. Si le créateur révoque les droits en cours de campagne, tu n'as aucun recours sans clause de préavis.

## Ce que ça donne côté performance

La question qui revient toujours : est-ce que ça vaut vraiment le setup supplémentaire par rapport à une pub UGC classique (créatif du créateur, diffusé depuis le compte marque) ?

Les données disponibles suggèrent que oui. À condition de comparer ce qui est comparable.

Une méta-analyse de 15 split tests conduits par Meta Marketing Science (2022) sur des annonceurs qui ont basculé leurs créatifs UGC en Partnership Ads observe un CTR supérieur de 53% et un CPA inférieur de 19% par rapport aux mêmes créatifs diffusés depuis le compte marque. Le mécanisme avancé : le compte créateur apporte son historique d'engagement propre, ce qui enrichit le signal de pertinence que l'algorithme utilise pour scorer l'annonce avant l'enchère.

Avec Andromeda, cet avantage est structurellement renforcé : l'algorithme lit à la fois le créatif et les signaux du compte depuis lequel la pub est diffusée. Un compte créateur avec un historique d'engagement fort donne à Andromeda un signal de qualité supplémentaire qu'un compte marque classique ne peut pas reproduire.

Ce n'est pas magique. Un mauvais créatif en Partnership Ad reste un mauvais créatif. Mais à qualité créative égale, la diffusion depuis un compte créateur actif donne statistiquement de meilleurs résultats.

<Callout variant="insight" title="Comment mesurer le ROAS des PA sans biais">
Si tes PA sont dans une ASC ou CBO, Andromeda redistribue le budget entre tous les créatifs. Le ROAS affiché par créatif est donc biaisé par la redistribution. La méthode pour mesurer l'impact réel des PA : test d'incrémentalité. Suspendre les PA pendant 7 à 14 jours à budget constant, comparer les résultats globaux du compte avant et après.
</Callout>

## Le process en 6 étapes résumé

Pour éviter les allers-retours avec le créateur et les blocages de dernière minute, voici l'ordre optimal :

1. **Signer le contrat créateur** avant de parler technique (droits, durée, révocation)
2. **Vérifier les prérequis** côté créateur (compte Pro/Créateur, Branded Content activé)
3. **Finaliser le créatif** (vidéo/image prête à uploader). Ne pas demander le code avant cette étape
4. **Demander le code** au créateur (LBC pour Instagram, CAA pour Facebook, ou les deux)
5. **Configurer le créatif** dans Ads Manager + entrer le code + injecter dans la campagne ASC existante
6. **Prévenir le créateur** de valider la notification Meta dès réception

Délai réaliste bout en bout, avec un créateur réactif : **2 à 4 jours ouvrés**.
